## 📌 Project Overview

This project focuses on **Exploratory Data Analysis (EDA)** to uncover the relationship between **social media usage** and **teen mental health**.  
The analysis identifies patterns related to depression, anxiety, stress, sleep quality, addiction levels, and academic performance.

The goal of this project is to transform raw data into meaningful insights that can help understand behavioral trends among teenagers.

---

## 🎯 Objectives

- Analyze social media usage patterns among teenagers  
- Identify factors affecting mental health  
- Explore depression distribution  
- Study sleep quality and its impact  
- Understand stress and anxiety levels  
- Analyze academic performance correlation  
- Generate meaningful visual insights  

---

## 📂 Dataset Information

The dataset includes:

- Age
- Gender
- Social Media Usage Hours
- Sleep Hours
- Stress Level
- Anxiety Level
- Addiction Level
- Academic Performance
- Social Interaction Score
- Physical Activity Hours
- Depression Label

## 📈 Key Insights

✅ Higher social media usage correlates with increased stress and anxiety  

✅ Poor sleep quality is strongly associated with depression  

✅ Higher addiction levels indicate higher depression risk  

✅ Academic performance is lower among depressed students  

✅ Physical activity contributes to better mental well-being
